Context-Aware React Component System

üìã Component Overview

This React component system enables seamless switching between Enterprise admin and Agency seat views within the same user session. The system maintains state during context switching and provides context-specific navigation, dashboards, and notifications.

üéõÔ∏è ContextSwitcher Component

A dropdown component that allows users to switch between different enterprise and agency contexts without losing their current session state.

Key Features:

  • Groups contexts by type (Enterprise vs Agency)
  • Search functionality for large context lists
  • Visual indicators for current context
  • Smooth animations and transitions
  • Responsive design for mobile devices
  • Dark mode support

üìä ContextAwareDashboard Component

Renders different dashboard layouts based on the current context - Enterprise Command Center for enterprise users and Agency Workflow Hub for agency users.

Enterprise Features:

  • Policy Management dashboard
  • Seat Management overview
  • Audit Center with real-time metrics
  • Analytics and compliance reporting

Agency Features:

  • Submissions tracking
  • Client Compliance monitoring
  • Tool Management interface
  • Quick Actions for common tasks

üß≠ ContextAwareNavigation Component

Provides context-specific navigation menus that adapt based on the user's current context and permissions.

Navigation Features:

  • Dynamic menu items based on context
  • Permission-based navigation
  • Badge indicators for notifications
  • Mobile-responsive sidebar
  • User profile and logout integration

üîî NotificationCenter Component

Aggregates notifications from all user contexts and provides filtering by context type and priority.

Notification Features:

  • Context-aware notification filtering
  • Priority-based visual indicators
  • Real-time notification updates
  • Mark as read functionality
  • Context badges for each notification
  • Time-based sorting and display

üéØ Usage Examples

Here are some common usage patterns for the context-aware components:

Basic App Integration:

Standalone ContextSwitcher:

Custom Dashboard Integration:

import ContextAwareDashboard from './components/ContextAwareDashboard.jsx'; function DashboardPage() { const handleNavigate = (route) => { // Handle navigation to different pages console.log('Navigating to:', route); }; return ( <ContextAwareDashboard currentContext={currentContext} onNavigate={handleNavigate} /> ); }
