API Endpoints

Seat Management

  • GET /api/enterprise/:id/seats - Get all seats
  • POST /api/enterprise/:id/seats - Create new seat
  • PUT /api/enterprise/:id/seats/:seatId - Update seat
  • DELETE /api/enterprise/:id/seats/:seatId - Delete seat

Policy & Analytics

  • POST /api/enterprise/:id/seats/bulk-policy-assignment - Bulk assign policies
  • GET /api/enterprise/:id/seats/analytics - Get seat analytics
  • GET /api/enterprise/:id/policies/available - Get available policies
  • GET /api/enterprise/:id/seats/:seatId/compliance-report - Get compliance report

Integration

The Seat Management Dashboard integrates seamlessly with the existing AICOMPLYR context switching system. Enterprise users can access seat management through the context switcher, and all seat operations are properly logged and tracked in the audit system.
